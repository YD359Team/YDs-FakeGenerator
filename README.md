# YDs FakeGenerator

Generate various data with this library!

## Features

- Use single class for any generator methods
- Create data with needed count

## Available types

### Boolean

- Boolean (true or false)

###	Numbers

- Integer numbers (i8, u8, i32, u32)
- Floating numbers (f32, f64)

### String

- FillMask (use any format like `+#(###)###-##-##` for phone number)
- Countries
- Capitals
- Languages
- Names (male, female, all)
- Persons (male, female, all)
- Prices (use any `CultureInfo`)

### Other

- Enum values
- Locations
